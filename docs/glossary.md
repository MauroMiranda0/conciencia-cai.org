# Glosario · conciencia-cai.org

- **Modelo Minnesota + 12 Pasos**: Enfoque terapéutico integral que combina evidencia clínica, espiritualidad y acompañamiento familiar; base metodológica del centro.
- **Sede Varonil**: Instalaciones exclusivas para hombres; se activa al seleccionar “hombres” o hashes `#sede-varonil`.
- **Sede Femenil**: Instalaciones exclusivas para mujeres; se activa mediante “mujeres” o `#sede-femenil`.
- **Coordinación clínica**: Equipo multidisciplinario que atiende mensajes y coordina valoraciones; aparece en copys y notas de canal.
- **Valoración confidencial**: Primera cita diagnóstica; principal CTA (“Agenda tu valoración”) enfocada en inmediatez y privacidad.
- **Micro-tonos del formulario**: Variantes `care` (contención emocional) y `sage` (orientación clara) usados para microcopy dinámico en `ContactSection`.
- **Modelo Minnesota PDF**: Documento descargable (`/docs/modelo-minnesota.pdf`) referenciado cuando el hash `#metodo` abre el modal o enlace externo.
- **FloatingNavButtons**: Accesos flotantes persistentes (WhatsApp y Modelo Minnesota); deben permanecer minimalistas y accesibles.
- **NormalizeSedeValue**: Helper que homologa entradas del usuario a `hombres` o `mujeres`; previene estados inconsistentes.
- **Theme Men/Women**: Clases de `<body>` que ajustan variables CSS (`--accent-color`, etc.) para diferenciar visualmente sedes.
